# 🖐️ Hand Gesture Controlled Particle System

A real-time hand gesture particle engine built with Python, Flask, MediaPipe, and HTML5 Canvas.

## ✨ Features
- 10 hand gestures detected using MediaPipe 0.10+
- JARVIS / Iron Man HUD aesthetic
- Real-time particle physics responding to gestures
- Angle-based finger detection with debounce for accuracy

## 🚀 Setup
pip install flask opencv-python mediapipe numpy
python app.py
# Open http://localhost:5000
